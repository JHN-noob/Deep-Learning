BCE Loss Function

이진 분류 문제에서는 BCE 손실함수를 주로 사용한다.
  • H(P, Q) = -(P(n) * logQ(n)를 2번 더한 값)
Regression 문제에서 MSE 손실함수를 사용하면 함수가 non-convex(로컬 미니마가 많음)할 수 있다.
예측값과 실제값이 같으면
  • loss = -(1 * log1 + (1 - 1) * log(1 - 1)) = 0
예측값과 실제값이 완전히 다르면
  • loss = -1log(0 + E) + (1 - 1)log(1 - 0) = 무한대
시그모이드 함수가 0~1 사이의 값을 출력하므로 이진 분류 문제에서 BCE 손실함수와 주로 사용한다.
